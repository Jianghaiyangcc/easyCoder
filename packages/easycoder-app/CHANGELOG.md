# Changelog

## Version 14 - 2026-04-27

本次更新重点修复终端绑定链路，并上线更安全的手机号登录恢复能力，让账号连接更稳更省心。

- 新增手机号验证码登录能力。
- 新增账户 Secret 备份恢复流程，降低换机或重装后的恢复成本。
- 修复终端连接页面可能一闪而过的问题。
- 优化 Web 绑定链接解析逻辑（支持 `?key=` 并兼容旧链接格式）。
- 修复未登录状态下确认终端连接可能出现的崩溃。
- 增加未登录提示，并在未认证时禁用“接受连接”。

## Version 13 - 2026-04-26

本次更新聚焦账户流转、语音配额反馈和设置页体验，减少卡顿、重试循环和状态不清晰问题。

- 新增手机号绑定（阿里云短信验证码）。
- 修复手机号相关流程中可能出现的重试循环与退出后重定向异常。
- 修复退出登录后页面不必要刷新，改为直接跳转首页。
- 优化平板登录后首页自动路由逻辑，减少错误跳转。
- 优化语音设置，按语音提供方展示用量与限制信息。
- 增强语音配额与限额反馈，超限时提示更明确。
- 优化支持入口在小屏下的布局，并补充付费墙错误提示。
- 本地化“设备恢复/绑定”相关页面文案。
- 精简设置页入口，移除不再使用的“已连接账户”聚合入口。
- 下线社交 Inbox 相关入口与同步面板，减少干扰项。

## Version 12 - 2026-04-25

本次更新完成品牌迁移并同步升级输入与会话创建体验，让新品牌下的整体操作路径更清晰。

- 完成产品品牌从旧命名到 EasyCoder 的整体迁移与对齐。
- 修复品牌迁移后设备元数据解析兼容问题，避免机器信息异常。
- 新增“快捷短语（Quick Phrases）”，支持常用消息一键发送。
- 优化输入区交互：发送/停止按钮合并，状态切换更直观。
- 优化模式选项布局，将关键模式操作放到输入区底部动作行。
- 支持将语音转写结果直接写入输入草稿，减少重复编辑。
- 支持在会话输入区内联选择权限模式/模型努力级别等关键参数。
- 优化设置页 Logo 在大屏设备下的缩放表现。
- 优化服务器配置保存流程与确认提示文案。
