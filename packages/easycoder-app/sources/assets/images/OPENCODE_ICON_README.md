# OpenCode 图标说明

## 当前状态
当前使用的是临时占位符图标（复制自 openclaw 图标）。

## 需要的图标文件
请准备 OpenCode 官方 logo（⌬ 符号），生成以下三个尺寸的 PNG 文件：

### 文件路径
- `packages/easycoder-app/sources/assets/images/icon-opencode.png` (128x128)
- `packages/easycoder-app/sources/assets/images/icon-opencode@2x.png` (256x256)
- `packages/easycoder-app/sources/assets/images/icon-opencode@3x.png` (384x384)

### 图标要求
- 格式：PNG
- 背景：透明
- 符号：⌬（OpenCode logo）
- 颜色：建议使用官方品牌色

## 获取官方 logo
可以从以下来源获取 OpenCode 官方 logo：
1. OpenCode 官网：https://opencode.ai
2. GitHub repo：https://github.com/opencode-ai/opencode

## 替换步骤
1. 下载或生成三个尺寸的图标文件
2. 替换上述三个占位符文件
3. 重新构建 App

## 参考图标尺寸
- icon-claude.png: 156x156
- icon-claude@2x.png: 312x312
- icon-claude@3x.png: 468x468
- icon-openclaw.png: 128x128
- icon-openclaw@2x.png: 256x256
- icon-openclaw@3x.png: 384x384

建议使用与 openclaw 相同的尺寸（128/256/384）。
